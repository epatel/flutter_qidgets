
Build widgets hierarchies quickly with qidgets.

## Features

Qidgets add shorthand quick access to build common widget hierarchies.

Example.
```
// qidgets
[
  'Hello'.wText,
  'world!'.wText,
].column.center,

// Normal
Center(
  child: Column(
    children: [
      Text('Hello'),
      Text('world!'),
    ],
  ),
),
```
